# 肖家胜 PPT 作品集网站｜克制清爽版

这一版根据你的反馈重新调整：

- 去掉复杂装饰、跑马灯、错位卡片
- 中文排版更规整
- 页面层级更清楚
- 作品展示更安静、专业

## 文件结构

- index.html：网站首页
- styles.css：网站样式
- script.js：菜单与作品图预览
- assets/portfolio.pdf：原 PDF 作品集
- assets/images/：作品集页面图片

## 上传到 GitHub Pages

1. 解压压缩包
2. 删除仓库里的旧版 index.html、styles.css、script.js 和 assets 文件夹
3. 上传新版全部文件
4. 点击 Commit changes
5. 等 1-2 分钟刷新网站
